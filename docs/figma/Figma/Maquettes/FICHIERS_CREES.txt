🚀 DÉMARRAGE RAPIDE
═════════════════════════════════════════════════════════════════════════════════
OPTION 1: DÉVELOPPEMENT LOCAL (5 minutes)
─────────────────────────────────────────
$ cd /home/jm/ISB/ISBibliotheque/doc/Figma/Maquettes
$ pnpm install                    # ou: npm install
$ pnpm dev                        # ou: npm run dev
→ http://localhost:5173/
OPTION 2: DÉPLOIEMENT DOCKER (5 minutes)
─────────────────────────────────────────
$ docker-compose up -d
$ docker-compose ps
→ http://localhost/
OPTION 3: DÉPLOIEMENT NGINX (15 minutes)
─────────────────────────────────────────
$ pnpm build
$ sudo cp -r dist/* /var/www/isb-portail/
$ sudo nginx -t && sudo systemctl restart nginx
→ http://votre-serveur/
